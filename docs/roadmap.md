# Roadmap

A lightweight, evolving roadmap from prototype to demo, then toward stretch goals. Dates are
intentionally omitted while the project is in the concept stage.

## Milestone 0 — Foundations (current)
- [x] Create repository and documentation structure.
- [x] Draft PRD v0.1 (vision, platforms, MVP mechanic, tech stack).
- [ ] Confirm repo name, visibility, and asset-license wording.

## Milestone 1 — Reveal prototype
- [ ] Stand up a Unity 6 + Meta XR SDK project targeting Quest 3/3S.
- [ ] Implement grabbing a model and holding a spray can / brush (Interaction SDK).
- [ ] Prototype 1–2 reveal techniques (texture-space and/or vertex-color) and compare feel + perf.
- [ ] Pick the reveal technique.

## Milestone 2 — One cozy model, end to end
- [ ] Bring in a Funes.World model; decimate/retopo and compress textures for Quest.
- [ ] Full color-reveal of a single model with spray + brush.
- [ ] Add cozy layer: ambient audio, lighting, gentle haptics, completion beat.

## Milestone 3 — Playable demo build
- [ ] Stable framerate on Quest 3/3S across a full model.
- [ ] First-time players complete the loop without instruction.
- [ ] Internal playtest; gather "is it calming/satisfying?" feedback.

## Milestone 4 — Evaluate stretch goals
- [ ] Assess PCVR / Pico 4 / Android XR / Apple Vision Pro via OpenXR.
- [ ] Investigate AR-glasses feasibility (XReal, Spectacles) — research only.

## Possible depth additions (post-MVP, if needed)
- [ ] Optional X-ray "where to paint" hint.
- [ ] Light sequence/goal overlay that preserves the freeform core.
